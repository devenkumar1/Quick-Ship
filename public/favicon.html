Create Favicon.ico for QuickShip

To create a favicon.ico file from the SVG above, follow these steps:

  1. Go to an online ICO converter like convertio.co/svg-ico
  2. Upload the SVG file (public/favicon.svg) or copy-paste the SVG code
  3. Convert it to ICO format
  4. Download the ICO file and save it as "favicon.ico" in the public folder

Make sure to include the following sizes in your ICO file: 16x16, 32x32, and 48x48 pixels for best compatibility.
